===== HABILITAR PVP PARA TODOS OS JOGADORES =====

PROBLEMA: Apenas operadores podem atacar outros jogadores.
Isso acontece quando o mundo tem PvP desabilitado ou restricoes de interacao.

===== SOLUCAO: EXECUTAR NO CONSOLE DO SERVIDOR =====

Execute estes comandos no console (um por vez):

1. Habilitar PvP globalmente:
gamerule pvp true

2. Desabilitar restricao de interacao entre jogadores:
gamerule disablePlayerInteraction false

3. Verificar se funcionou:
gamerule pvp
gamerule disablePlayerInteraction

Deve mostrar:
- pvp = true
- disablePlayerInteraction = false

===== SE AINDA NAO FUNCIONAR =====

O problema pode estar no server.properties. Verifique se tem:

disable-player-interaction=false

Se estiver "true", mude para "false" e reinicie o servidor.

===== TESTAR =====

1. Conecte 2 jogadores normais (NAO operadores)
2. Um tenta atacar o outro
3. Deve funcionar normalmente
