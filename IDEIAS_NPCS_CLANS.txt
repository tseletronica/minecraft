===== NPCS COM CLANS - POSSIBILIDADES =====

SIM! E possivel criar NPCs (sentinelas, soldados, guardas) para cada clan!

===== OPCAO 1: USAR MOBS EXISTENTES =====

Podemos usar mobs do Minecraft e dar tags de clan para eles:

EXEMPLOS:
- Iron Golem = Guardiao do clan
- Villager = Comerciante do clan
- Wolf (lobo domesticado) = Soldado do clan
- Snow Golem = Sentinela do clan

VANTAGENS:
+ Facil de implementar
+ Ja tem IA pronta
+ Pode atacar inimigos automaticamente

DESVANTAGENS:
- Visual limitado (nao da pra customizar muito)
- Comportamento limitado

===== OPCAO 2: CRIAR ENTIDADES CUSTOMIZADAS =====

Criar NPCs completamente customizados com behavior pack:

EXEMPLOS:
- Soldado RED (armadura vermelha, espada)
- Arqueiro BLUE (arco, armadura azul)
- Mago GREEN (cajado, efeitos verdes)
- Cavaleiro YELLOW (armadura dourada, escudo)

VANTAGENS:
+ Visual 100% customizado
+ Comportamento customizado
+ Pode ter habilidades especiais

DESVANTAGENS:
- Mais complexo de criar
- Precisa de modelos 3D e texturas

===== OPCAO 3: USAR NPC ENTITY (MAIS SIMPLES) =====

Minecraft tem uma entidade "NPC" nativa:

FUNCIONALIDADES:
- Pode ter dialogo
- Pode executar comandos
- Pode ter skin customizada
- Pode ter nome colorido do clan

VANTAGENS:
+ Nativo do Minecraft
+ Facil de spawnar
+ Pode ter dialogos

DESVANTAGENS:
- Nao ataca (e pacifico)
- Nao se move sozinho

===== OPCAO 4: SISTEMA HIBRIDO (RECOMENDADO) =====

Usar mobs existentes + scripts para comportamento de clan:

EXEMPLO - GUARDIAO DE CLAN:
1. Spawnar Iron Golem
2. Dar tag do clan (clan_red, clan_blue, etc)
3. Script faz ele:
   - Atacar jogadores de outros clans
   - Proteger jogadores do mesmo clan
   - Ficar em area especifica (base do clan)

EXEMPLO - SOLDADO DE CLAN:
1. Spawnar Zombie/Skeleton customizado
2. Dar tag do clan
3. Dar armadura colorida do clan
4. Script faz ele:
   - Atacar apenas inimigos (outros clans)
   - Nao atacar aliados (mesmo clan)
   - Patrulhar area

===== O QUE VOCE QUER? =====

1. NPCs SIMPLES (dialogos, comercio)?
   -> Usar entidade NPC nativa

2. GUARDAS/SENTINELAS (defesa de base)?

4. NPCS CUSTOMIZADOS (visual unico)?
   -> Criar entidades customizadas (mais complexo)
