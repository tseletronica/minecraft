[ATENÇÃO] COMO ATUALIZAR O SERVIDOR MINECRAFT SEM PERDER O SISTEMA DE CLÃS

Quando sair uma nova versão do Minecraft Bedrock Dedicated Server, NÃO delete esta pasta e nem substitua tudo de uma vez. Siga este passo a passo para não perder seu mundo nem o sistema de clãs:

1. Baixe a nova versão do servidor (arquivo .zip).
2. Extraia o conteúdo em uma NOVA pasta (ex: "bedrock-server-novo").
3. Copie as seguintes pastas DO SEU SERVIDOR ANTIGO para o NOVO, substituindo se necessário:
   
   a) A pasta "worlds" inteira (aqui ficam as construções e o inventário dos jogadores).
   b) A pasta "behavior_packs/clan_system" (aqui fica o código do Clã que criamos).
   c) O arquivo "server.properties" (suas configurações).
   d) O arquivo "allowlist.json" e "permissions.json" (se você usar whitelist/permissões).

4. Abra o novo servidor e verifique se tudo carregou.

IMPORTANTE:
Sempre faça um BACKUP (cópia) da pasta "worlds" e da pasta "behavior_packs/clan_system" antes de qualquer alteração!

Se você apenas arrastar os arquivos novos para cima da pasta antiga, pode corromper ou substituir arquivos importantes. Use o método de criar uma pasta nova e copiar SEUS dados para lá.
