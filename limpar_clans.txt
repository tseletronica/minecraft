===== COMANDOS PARA LIMPAR TODOS OS JOGADORES =====

Execute estes comandos no console do servidor (um por vez):

tag @a remove clan_red
tag @a remove clan_blue
tag @a remove clan_green
tag @a remove clan_yellow

===== VERIFICAR SE LIMPOU =====

Para verificar se todos foram removidos:

tag @a list

Nenhum jogador deve ter tags de clan.

===== REINICIAR SERVIDOR =====

Depois de limpar, reinicie o servidor para garantir que tudo funcione corretamente.
