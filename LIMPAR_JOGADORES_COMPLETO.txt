===== COMO LIMPAR TODOS OS JOGADORES (INCLUINDO OFFLINE) =====

PROBLEMA: O comando "tag @a remove" so funciona para jogadores ONLINE.
Jogadores offline mantem as tags salvas no banco de dados do mundo.

===== SOLUCAO 1: RESETAR MUNDO (MAIS SIMPLES) =====

Se voce nao se importa em perder o progresso do mundo:

1. PARAR o servidor
2. APAGAR a pasta: worlds/Bedrock level
3. COPIAR um mundo novo (criado no cliente com Beta APIs ativado)
4. INICIAR o servidor

Todos os jogadores vao escolher clan novamente.

===== SOLUCAO 3: SCRIPT AUTOMATICO =====

Adicione este codigo temporario no main.js para limpar automaticamente
quando cada jogador entrar:

// CODIGO TEMPORARIO - REMOVER DEPOIS
world.afterEvents.playerSpawn.subscribe((event) => {
    const player = event.player;
    if (!event.initialSpawn) return;
    
    // Remover todas as tags de clan
    player.removeTag('clan_red');
    player.removeTag('clan_blue');
    player.removeTag('clan_green');
    player.removeTag('clan_yellow');
    
    player.sendMessage('Â§eSuas tags de clan foram resetadas!');
});

Depois que todos os jogadores entrarem uma vez, REMOVA este codigo.
